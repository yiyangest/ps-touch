Guide de connexion des applications tactiles pour
Adobe® Photoshop®
  

Résumé
Ce document vous explique comment connecter
votre application tactile Photoshop au logiciel Adobe® Photoshop® CS5 ou version ultérieure exécuté sur votre bureau.

Tout d’abord...

Vérifiez que vous utilisez Adobe Photoshop CS5 12.0.4 ou une version ultérieure.

  1. Accédez à Photoshop > A propos de Photoshop... (Mac) ou Aide > A propos de Photoshop (Windows) pour connaitre le numero de version.

  2. Si la version exécutée n’est pas 12.0.4 ou une version ultérieure, vous pouvez installer la dernière version en choisissant Aide > Mises à jour... dans Photoshop.

Ensuite...

Activez les connexions à distance dans Photoshop.

  1. Lancez Photoshop.

  2. Dans le menu Edition, sélectionnez Connexions à distance...

  1. Cochez la case Activer les connexions à distance.

  2. Créez un mot de passe personnalisé, puis cliquez sur OK pour établir une connexion.

Puis...

Connectez votre application tactile Photoshop à Photoshop à l’aide de l’option de découverte de service.

  1. Lancez votre application tactile Photoshop.

  2. Touchez l’icône PS dans l’angle inférieur droit.

  1. Sélectionnez le nom du service tel que défini dans la boîte de dialogue Connexions à distance de Photoshop.

  1. Entrez le mot de passe et touchez Connecter.

  1. Une fois la connexion établie, l’icône PS située dans l’angle inférieur droit devient bleue.

Vous devriez maintenant être connecté à Photoshop.

Option d’adresse IP manuelle

Connectez votre application tactile Photoshop à Photoshop.

  1. Lancez votre application tactile Photoshop.

  2. Touchez l’icône PS dans l’angle inférieur droit.

  1. Touchez Nouveau.

  1. Saisissez manuellement l’adresse IP et le mot de passe figurant dans la boîte de dialogue Connexions à distance, puis touchez Connecter.


  1. Une fois la connexion établie, l’icône PS située dans l’angle inférieur droit devient bleue.

Vous devez maintenant être connecté à Photoshop.
