Text color
Lets give the text some color.
Select and un-hide "FIRE copy" layer.

Click on "Lock Transparent Pixel"
(it looks like checkers on the Layer panel)


 Apply the gradient by dragging from top of image to bottom.
Last, bring up the Layer Style blend mode

Layer -> Layer Style -> Blending Options
Click on "Drop Shadow" and change the color to R: 240, G:100, B:0
(this will give an orange like color.Drop the Opacity down to about 60%. Change the distance and Spread to 4. Change the size to 10.)


Select the gradient tool. Change the gradient type to "Orange Yellow Orange"


 Click on "Inner Shadow" and pick this
redish color: R:230, G:0, B:0.
Change the Distance to 10, Choke to 0, and Size to 60.
