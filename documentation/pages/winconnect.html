Windows Connection How-To

A typical user workflow would be something like this:

  1. Install Photoshop CS5.
  2. Enable Remote connections. Edit > Remote Connectionsâ€¦
  3. In the Remote Connections dialog, set Password and check Enable Remote Connections. Click OK.
  4. Windows will throw a dialog asking if it should allow Photoshop CS5 through the firewall.
  5. Click OK.

Windows 7 Dialog shown

If a user clicks No, they have basically just told the OS not to allow communication with remote apps. Even though everything looks OK, the OS will block any communication.

Steps to Resolve issue

  1. Windows Start Menu > Control Panel
  2. Go to Firewall Settings in Control Panel
  3. Click on "Allow a program through Windows Firewall". This option is slightly different for each OS.
  4. If is listed in the list of programs simply. Click On check box to allow to communicate through Firewall. User may have to enter admin password to make change.

    On Windows 7 they differentiate between Work, Home, Public networks. User will have to make their own choice there. I would recommend Work and Home to start. I do not envision too many people using thier apps in Starbucks but you never know. Just something to be aware of.
  5. If is *NOT* listed in the list of programs.
  6. Click "Add Program"
  7. Choose from list of Programs. May require Admin Password.
  8. OK all dialogs.
  9. Restart to be safe.
