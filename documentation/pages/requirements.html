System and Software Requirements

  • Adobe Photoshop CS5

    is required as the Photoshop server. In addition, Adobe Photoshop CS5 with the 12.0.4 update can also be configured as a Photoshop server.

  • Adobe Flash Builder "Burrito"

    is required for creating client applications using the AIR runtime. AIR runtimes are available on mobile devices and desktop.

  • Google Android SDK

    is required for creating Android applications for Android devices. The Android examples in this SDK use the Galileo version of Eclipse with the Android 2.2 version of the Android SDK.

  • Apple iOS Dev Center

    is required for creating iOS or desktop applications for the Apple platform. The iOS examples in this SDK use the Xcode 3.2.5 version that comes with the iOS 4.2 SDK from Apple.

  • Bonjour

    is required for discovering the Photoshop server on the Windows platform. Users will need to download and install the service from the Apple website.
