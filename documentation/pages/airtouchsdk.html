PhotoshopTouchSDK version 0.5

API and Architecture: Daniel Koestler (http://blogs.adobe.com/koestler)
Encryption and Architecture: Renaun Erickson (http://renaun.com)

For more information on the API, please visit the Photoshop Touch SDK's ActionScript 3 API Document.

Required libraries

  • as3corelib (https://github.com/mikechambers/as3corelib)
  • as3crypto (https://code.google.com/p/as3crypto/)
  • StructuredLogTestingSDK (https://code.google.com/p/structuredlogtesting/)

Overview

If you're compiling the PhotoshopTouchSDK yourself, you can either include the source paths from the above libraries, or drop the SWC files into the "libs" folder of your project.

Code is in src/com/adobe/photoshop/
SWC is bin/PhotoshopTouchSDK.swc
